## 项目说明
An Express-MySQL User Base App

## 启动服务器
~~~
npm run start
~~~

## 项目依赖和实现 
### 模板引擎
handlerbars
### 数据库
mysqljs/mysql
### node环境配置
dotenv 
### 表单验证
通过 express-validator 实现后台表单验证.  
### hash user password
bcrypt
### session
express-session
### session store
express-mysql-session
### anthentication
passport & passport-local
### process manager
pm2
